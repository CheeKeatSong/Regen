# Regen - MEN Stack web application
Re = Requirement

gen = Generation

Regen is requirement engineering platform to address the tedious requirement elicitation process by automating it with capabilites of chatbot.

## Features

* **Project Management**. A dashboard is available to manage every project, such as stakeholders in a project, information and so on.

* **Elicitation**. Relevant users to a project can provide requirement information to the chatbot.

* **User Profile**. Player can manage their profile information.
